# AanTan

A Flutter app with Android home screen widget for dual users with customizable themes, images, and update history.

## Quick Start

```bash
flutter pub get
flutter run
```

## Testing

```bash
flutter test
```

73 unit tests covering all services and models with mocked database dependencies.
